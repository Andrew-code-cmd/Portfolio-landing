Простой лендинг на Next.js для индивидуального разработчика

## Стек
[![Next.js](https://img.shields.io/badge/Next.js-15.0-black?logo=next.js&logoColor=white)](https://nextjs.org)
[![React](https://img.shields.io/badge/React-18.2-61DAFB?logo=react&logoColor=black)](https://react.dev)
[![Tailwind](https://img.shields.io/badge/Tailwind-3.4-06B6D4?logo=tailwindcss&logoColor=white)](https://tailwindcss.com)

> **Минималистичный, быстрый и адаптивный лендинг для фрилансера или fullstack-разработчика.**  
> Проект создан для демонстрации навыков, услуг, портфолио и контактов

##  Особенности

- **Скорость** — Next.js App Router 
- **Tailwind CSS** — без лишнего CSS
- **Адаптив** — под мобильные устройства 
-  **Форма обратной связи** — с валидацией данных
